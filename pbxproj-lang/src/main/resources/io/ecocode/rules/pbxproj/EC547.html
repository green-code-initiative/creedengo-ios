Dark Mode was introduced in iOS 13. It allows the user to switch its system to a dark mode and the apps inherits it automatically by default. Yet is possible to overrides it to light mode by setting the value Light for the key UIUserInterfaceStyle in the Information Property List. This should be avoided since displaying dark colors has a significant impact on energy consumption for mobile devices with OLED screens. The value for the key UIUserInterfaceStyle must be Automatic or this key must be removed from the Information Property List.
For more information, please refer to the official Apple documentation .
